# Output Data

## `session_counts.tsv`

Generated by `invoke run-statistics`. Long-format table with one row per (dataset, subject) pair.

Columns: `dataset`, `subject`, `n_sessions`

## `fmri_stats.tsv`

Generated by `invoke run-fmri-stats`. One row per dataset with aggregate fMRI acquisition statistics.

Columns: `dataset`, `total_runs`, `avg_runs_per_session`, `avg_run_duration_h`, `avg_session_duration_h`, `total_duration_h`

Duration columns are in hours. Cells are empty (NaN) when volume counts cannot be determined from sidecar JSON (e.g. `emotion-videos`).

## `fmri_stats.json`

BIDS-style JSON sidecar for `fmri_stats.tsv`. Describes each column with a `Description` and `Units` field.

## `figure_sessions.png`

Generated by `invoke run-notebooks` from `notebooks/figure_sessions.ipynb`.

Grid of bar charts: one summary panel (total sessions per subject across all datasets) and one panel per dataset showing the per-subject session count.
