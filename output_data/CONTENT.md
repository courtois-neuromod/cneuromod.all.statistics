# Output Data

## `session_counts.tsv`

Generated by `invoke run-statistics`. Long-format table with one row per (dataset, subject) pair.

Columns: `dataset`, `subject`, `n_sessions`

## `fmri_stats.tsv`

Generated by `invoke run-fmri-stats`. One row per dataset with aggregate fMRI acquisition statistics.

Columns: `dataset`, `total_runs`, `avg_runs_per_session`, `avg_run_duration_h`, `avg_session_duration_h`, `total_duration_h`

Duration columns are in hours. Cells are empty (NaN) when volume counts cannot be determined from sidecar JSON (e.g. `emotion-videos`).

## `fmri_stats.json`

BIDS-style JSON sidecar for `fmri_stats.tsv`. Describes each column with a `Description` and `Units` field.

## `figure_sessions.png`

Generated by `invoke run-notebooks` from `notebooks/figure_sessions.ipynb`.

Grid of bar charts: one summary panel (total sessions per subject across all datasets) and one panel per dataset showing the per-subject session count.

## `fmri_stats_per_subject.tsv`

Generated by `invoke run-fmri-per-subject-stats`. Long-format table with one row per (dataset, subject) pair.

Columns: `dataset`, `subject`, `total_runs`, `avg_runs_per_session`, `avg_session_duration_h`, `total_duration_h`

Duration columns are in hours. Cells are empty (NaN) when volume counts cannot be determined (e.g. `emotion-videos`, or subjects with no runs in a dataset).

## `figure_fmri_stats.png`

Generated by `invoke run-notebooks` from `notebooks/figure_fmri_stats.ipynb`.

Grid of bar charts with one panel per fMRI metric: total runs, avg runs per session, avg run duration (h), avg session duration (h), and total duration (h). Each panel shows all datasets on the x-axis.

## `figure_fmri_per_subject_total_runs.png`, `figure_fmri_per_subject_total_duration_h.png`, `figure_fmri_per_subject_avg_session_duration_h.png`

Generated by `invoke run-notebooks` from `notebooks/figure_fmri_per_subject.ipynb`.

One figure per metric. Each figure has a summary panel (average across datasets per subject) and one panel per dataset, showing per-subject bars — mirroring the layout of `figure_sessions.png`.

## `figure_fmri_total_hours.png`

Generated by `invoke run-notebooks` from `notebooks/figure_fmri_hours.ipynb`.

Single bar chart showing the total fMRI acquisition time (hours) per dataset, aggregated across all subjects.

## `figure_fmri_hours_min_avg_max.png`

Generated by `invoke run-notebooks` from `notebooks/figure_fmri_hours.ipynb`.

Grouped bar chart showing, for each dataset, the min / avg / max of `total_duration_h` across subjects — computed only over subjects who have data in that dataset (i.e. `total_runs > 0`).
